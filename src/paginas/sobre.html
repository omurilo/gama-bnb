Hiring Coders Challenge

Esta aplicação foi feita para cumprimento de um desafio do Hiring Coders da Gama Academy, que consistia em criar uma "cópia" do airbnb, porém, não gosto de criar cópias fiéis então dei um toque diferente ao visual do site. Foi utilizado para criação deste site somente os conceitos básicos de Javascript, os que foram apresentados até aqui (09/05/2020) no Hiring Coders, repositório do projeto, e por isso não possui padrões de projeto e design patterns, é tudo bem simples só com um webpack para geração e minificação dos arquivos para produção.
